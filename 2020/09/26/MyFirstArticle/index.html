    Markdown 常用语法

    Luminous' blog
    2020-09-26

    第一篇测试博客，用于熟悉Markdown语法
    （注意：标记后面要加空格再填写内容，显示样式与自身主题样式有关）

    Markdown 标题

    使用 = 和 - 标记一级和二级标题

    显示效果如下：

    我是一级标题

    我是二级标题

    使用 # 标记

    显示效果如下：

    我是一级标题

    我是二级标题

    我是三级标题

    Markdown 列表

    使用 * + - 作为无序列表标记

    显示效果如下：

    • 第一项
    • 第二项
    • 第一项
    • 第二项
    • 第一项
    • 第二项

    使用数字加上.表示有序列表

    显示效果如下：

    1. 第一项
    2. 第二项
    3. 第三项

    列表嵌套

    在子列表的选项面前加4 个空格

    显示效果如下：

    1. 第一项
      • 第一项嵌套的第一个元素
      • 第一项嵌套的第二个元素
    2. 第二项
      • 第二项嵌套的第一个元素
      • 第二项嵌套的第二个元素

    Markdown 区块

    在段落开头使用 > 符号，多个表嵌套

    显示效果如下：

    区块引用

    菜鸟教程
    学习学习学习

    Markdown 链接

    显示效果如下：
    这是一个链接 Markdown 菜鸟链接

    Markdown 图片

    语法格式：

    • 开头一个感叹号!
    • 接着一个方括号，里面放上图片的代替文字
    • 接着一个圆括号，里面放上图片的网址，最后还可以用引号包住加上选择性的 title 属性的文字

    显示效果如下：

    Markdown 还没有办法指定图片的高度宽度，如果需要的话可以使用普通 HTML 的 img 标签
    Markdown 目前支持的 HTML 元素有：kbd, b, i, em, sup, sub, br 等

    以上为菜鸟教程的部分内容，更多高级技巧请前往 Markdown 的菜鸟教程查看学习
