昨天晚上刚在线上参加的“爱学习”教育集团校招前端工程师的笔试题，趁着还热乎的，来记录总结一下笔试题的内容，查漏补缺总结经验。题目是我根据我的草稿本回忆的，考试的时候没记录太全，所以题干只是大概意思并非原题，选择题选项也没记全，只给最终答案。

单选 15 道

1. 下面代码运行结果

答案：0 1 0

这题考察的是闭包相关的知识，关于闭包我一直没怎么理解，查阅一番资料再加上朋友指导之后今天来简单的总结一下。

  • 什么是闭包？
    红宝书对闭包的定义是指有权访问另一个函数作用域中的变量的函数，常见的创建闭包方式就是在函数内部创建另一个函数，内部的函数就形成了一个闭包，本题中 function () {console.log(i++);}; 就是一个闭包。
  • 闭包的作用
    1. 可以在函数的外部访问到函数内部的局部变量
    2. 让这些变量始终保存在内存中，不会随着函数的结束而自动销毁

在 JavaScript 中，正常情况下，在函数的外部是不能访问函数内部的变量的，而且随着函数运行结束，函数内部定义的变量也会被自动回收机制回收掉。但是在闭包中，就可以不受限制。

闭包和作用域链放在一起看

当代码在一个环境中执行时，会创建变量对象的一个作用域链（scope chain）。作用域链的用途，是保证对执行环境有权访问的所有变量和函数的有序访问。作用域链的前端，始终都是当前执行的代码所在环境的变量对象。如果这个环境是函数，则将其活动对象（activation object）作为变量对象。活动对象在最开始时只包含一个变量，即 arguments 对象（这个对象在全局环境下是不存在的）。作用域链中的下一个变量对象来自包含（外部）环境，而再下一个变量对象则来自下一个包含环境。这样，一直延续到全局执行环境；全局执行环境的变量对象始终都是作用域链中的最后一个对象。

这段长文字是红宝书中对作用域链的解释。我个人的通俗理解就是，在一个函数里能访问的变量按照一定的层级关系会形成一个有序的链子，最先能访问的变量也就是链子上最先能接触到的地方，是当前函数的活动对象（就是函数里定义或者重新赋值的变量），下一个能访问的变量就是链子的下一个点，就是当前函数的包含环境（个人感觉称之为外部环境更容易理解），然后一级一级往下找，找到最外面的全局执行环境为止。

理解了作用域链再回过来看闭包，我们知道函数 A 内部的函数 B 可以访问其外部环境即函数 A 的内部变量，那么如果将函数 B 作为 return 返回值返回，并在函数 A 的外部用一个变量 C 来接收这个返回值，那么我们在函数 A 外部操作变量 C 时，实际上就是在操作函数 A 的返回值函数 B，而函数 B 是可以访问函数 A 的内部变量的，这样我们就做到了在函数 A 外部通过变量 C 访问函数 A 内部的变量。这是闭包的第一个作用。因为有了这个作用，所以也就有了第二个作用：让这些变量始终保存在内存中，不会随着函数的结束而自动销毁。意思就是函数 A 内部的变量会和函数 A 外部的变量 C 一同存在，不会因为函数 A 执行完毕而被垃圾回收机制回收。

哇，有没有觉得突然能理解闭包了呢，反正我是有种恍然大悟的感觉，嘿嘿。

然后我们再来看这道题目，函数 Foo() 的返回值 函数 function () {console.log(i++);} 被赋给了变量 f1 和 f2，f1 第一次执行的时候，很显然就是打印 i++ 的返回值 0，这里再对 i++ 和 ++i 做个补充说明，i++ 是先返回值，再自增，++i 是先自增，再返回值。然后 f1 在执行第二次的时候，我们说过，函数 Foo() 内部的变量 i 是不会被垃圾回收机制回收的，因为它和变量 f1 一同存在，所以此时的 i 为 1，打印结果为 1。而 f2 开始执行的时候是跟 f1 没有任何关系的，它就等同于 f1 第一遍执行。所以最后的打印结果为 0 1 0。

3. CSS3 中的 translate 会改变 z 轴方向的值吗？

这题我不太清楚题干意思，因为很显然 CSS3 中对于 translate 是新增了 3D 变换的，可以改变对象 z 轴方向的值，但是对于 translate() 本身是只能设置 x, y 的值，translateZ() 或者 translate3d() 才可以设置 z 的值。这题我最终选择了会。

4. 设置新窗口打开的属性值为？

答案：_blank

其他属性值：

  1. _self：默认，在相同的框架中打开被链接文档
  2. _parent：在父框架集中打开被链接文档
  3. _top：在整个窗口中打开被链接文档
  4. framename：在指定的框架中打开被链接文档

6. nodejs 中的 process 是什么变量？

8. 以下哪个方法能判断数组 [1,2,”3”] 是否含有 “3” ？

答案：A

因为这题不太清楚，所以记了所有选项，排除很熟悉的 push()。

  1. some()：对数组中的每一项运行给定函数，如果该函数对任一项返回 true，则返回 true
  2. every()：对数组中的每一项运行给定函数，如果每一项都返回 true，则返回 true
    • 这两个方法很相似，都用于查询数组中的项是否满足某个条件，所以本题可以使用 some() 来实现
  3. concat()：可以基于当前数组中的所有项创建一个新数组，将接收到的参数添加到原数组的末尾返回新构建的数组

对数组的其他方法暂时不做赘述。
