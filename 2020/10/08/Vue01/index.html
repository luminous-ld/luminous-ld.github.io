Vue的常用指令笔记

Luminous' blog
2020-10-08

v-text / v-html

  • v-text

    • 设置标签的文本值( textContent )
    • 标签内的所有内容都会被 v-text 所替代
    • 无论内容是什么只会被解析为文本
  • v-html
    • 设置元素的 innerHTML
    • content 中有 html 结构会被解析为标签

v-on

  • 为元素绑定事件
  • 两种写法：v-on:click => @click
  • 方法绑定在 Vue 实例中的 methods 属性中

Vue 一般不需要考虑操作 DOM ，而是通过改变数据来改变页面内容，只要在方法中通过 this 来获取数据对象即可

  • 事件绑定的方法写成 函数调用 的形式可以传递自定义参数
  • 事件的后面跟上 .修饰符 可以对事件进行限制
  • 如 .enter 可以限制触发的按键为回车
  • 常见修饰符文档地址 https://cn.vuejs.org/v2/api/#v-on

v-show / v-if

  • 二者都是根据表达式值的真假，切换元素的显示和隐藏
  • v-show 是操纵样式 display 的值，作用的元素对象是一直存在的
  • v-if 是直接操纵 DOM 元素，隐藏的元素会直接从 DOM 树中删除
  • 实际开发中，如果需要频繁切换显示模式的元素我们使用 v-show，反之，前者消耗性能低

v-for

  • 根据数据生成列表结构
  • 语法结构 v-for=”item in arr”
  • 该指令是把作为模板的那个标签及内部所有内容进行拷贝

arr 为数组类型时

arr 为对象类型时

v-bind

  • 设置元素的属性，如 src, title, class 等
  • 语法结构 v-bind:属性名=表达式，可简写为 :属性名=表达式
  • class 的表达式有两种写法：
    • 三元表达式
    • 对象方式 {对象名:值}
  • 需要动态增删的 class 建议使用对象的方式

v-model (双向数据绑定)

  • 获取和设置表单元素的值
  • 绑定的数据会和表单元素值 value 相关联
  • 双向绑定，无论修改谁，另外一个都会跟着改变
