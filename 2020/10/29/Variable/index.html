ES 变量

Luminous' blog
2020-10-29

JavaScript 的变量和其他语言的变量有很大的区别

复制变量值

复制基本类型值

输出 5，num2 和 num1 是完全独立的，修改 num1 不会影响 num2

复制引用类型值

输出 Ruth, Mike，复制的副本实际上是一个指针，两个变量指向存储在堆中的同一个对象，所以修改值会影响

传递参数

ES 中所有函数的参数都是按值传递的，所以传递参数和变量复制一样

传递基本类型参数

输出 20, 30， count 作为参数传给函数，num 复制 count 的值，num 值在函数作用下改变但不会影响 count 值

传递引用类型参数

输出 undefined, Nicholas，证明是按值传参，而不是按引用传参，在函数中新定义的对象 obj 是一个局部对象，在函数执行完毕后就销毁，对结果不会有影响

检测类型

typeof

typeof 操作符是检测基本数据类型的最佳工具

  • 确定变量是字符串、数值、布尔值还是 undefined
  • 如果是对象或者 null，返回 object
  • 对象是函数，返回 function
  • Safari 5 及 之前和 Chrome 7 及之前版本浏览器用 typeof 检测正则表达式，也返回 function
  • IE 及 Firefox 对正则表达式返回 object

instanceof

instanceof 操作符检测对象是什么类型的对象
